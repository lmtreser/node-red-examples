# Node-RED Examples

Ejemplos varios para aprender y trabajar con [Node-RED](https://nodered.org/).
Para importarlos seguir: `Menu` &rarr; `Import` &rarr; `Paste flow json` o `Select file to import`.

Tildar la opción `Import to` `current flow` o `new flow`.

## Contenidos

- `example-01` Integración UI, MQTT y archivos (json, csv).
- `example-02` Agrega al ejemplo anterior imágenes y charts.
- `example-03` Comunicación con MySQL.
- `example-04` Ejemplo completo sistema de domótica.
- `example-05` Integración MQTT y UI.
